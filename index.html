테미 · 스탬프 랠리 MVP

키오스크·모바일 화면은 Spring Boot 정적 리소스로 제공됩니다. Android Studio 기준으로 서버를 띄운 뒤 아래 링크를 브라우저에서 여세요.

  1. File → Open → 이 저장소의 stamp-rally-server 폴더 선택 (pom.xml 있는 폴더)
  2. Maven 가져오기가 끝날 때까지 대기
  3. StampRallyApplication.java에서 Run (또는 Maven → spring-boot:run)

터미널만 쓸 경우: cd stamp-rally-server && mvn spring-boot:run

  • 테미 메인 (1920×1200 권장)
  • 모바일 입장
  • 스탬프 QR 4종 (현장 배치 데모)

저장소 루트의 예전 css/style.css·index.html은 파일로 직접 열 때용이며, API 연동은 위 주소에서 확인하는 것이 맞습니다.
